
# New Year Countdown

A small web project that shows a countdown to the New Year.

## Overview

This project contains a simple HTML/CSS/JavaScript countdown you can open in any modern browser.

## Files

- `index.html` — main page
- `style.css` — styles
- `script.js` — countdown logic
- `README.md` — this file

## Usage

1. Open `index.html` in your browser (double-click or drag into a browser window).
2. Or, serve the folder with a simple HTTP server, e.g.:

```
# Python 3
python -m http.server 8000

# Then open http://localhost:8000 in your browser
```

## Notes

- Works in modern desktop and mobile browsers.
- Edit `script.js` to change target date or appearance.
